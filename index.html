HTTP Status Codes (SQLite, CSV)

Contains list of HyperText Transfer Protocol (HTTP) response status codes in SQLite & CSV formats. It contains most of the commonly used codes. Code descriptions are taken from Wikipedia & IETF.

Database Structure

  • Key (Primary Key)
  • Code
  • Title
  • Summary
  • Wikipedia Description
  • Wikipedia Link
  • IETF Description
  • IETF Link
  • Class
